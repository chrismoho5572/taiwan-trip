# Moss Travel Planner Skill 🌿

สร้างแผนทริปพร้อมเว็บแอพให้ผู้ใช้

## โครงสร้าง

```
moss-travel-planner/
├── SKILL.md           ← คำแนะนำหลัก
├── assets/
│   ├── index.html     ← เทมเพลต HTML
│   ├── style.css      ← CSS พร้อมใช้
│   └── app.js         ← JavaScript logic
└── references/
    └── examples.md    ← ตัวอย่างข้อมูล
```

## วิธีใช้

1. **ถามข้อมูลจากผู้ใช้**
   - ไปไหน? กี่วัน? งบเท่าไร?
   - สนใจอะไร? (อาหาร, ช้อปปิ้ง, ถ่ายรูป)

2. **ค้นหาข้อมูล**
   - สถานที่ท่องเที่ยว
   - เวลาเปิด-ปิด
   - จุดถ่ายรูป
   - รูปภาพจาก Unsplash

3. **สร้างไฟล์**
   ```
   destination-trip/
   ├── index.html
   ├── style.css
   ├── app.js
   ├── data/places.json
   └── README.md
   ```

4. **Deploy**
   - Push ไป GitHub
   - เชื่อม Cloudflare Pages
   - URL พร้อมใช้!

## Key Features

- ✅ Real-time open/close status
- ✅ Day-by-day itinerary
- ✅ Photo spots with reference images
- ✅ Filter by place type
- ✅ Responsive design
- ✅ Free deployment
