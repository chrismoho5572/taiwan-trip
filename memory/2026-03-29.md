## 🎨 Design Lessons

- Avoid AI slop (Inter font, purple gradients, Space Grotesk)
- Use distinctive fonts (Playfair Display + Sarabun for Thai)
- Warm romantic colors (Coral + Cream + Gold) better than purple gradients
- Magazine-style layout more personal than generic travel sites
- Custom fonts require Google Fonts preconnect

- Revoke tokens immediately if accidentally shared
- Use Personal Access Tokens, not passwords
- Token scopes: only need "repo" for pushing
